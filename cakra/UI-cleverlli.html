Website Features:

Website Features:

Engaging Homepage: Utilizes parallax scrolling to create an interactive experience that showcases Cleverlli's creative atmosphere.

  • Clear Information Architecture: Separate pages for About, Pricing, and Contact, easily accessible via a top navigation bar, ensure users can find the information they need quickly.

  • Online Booking System: Integrated time picker allows visitors to schedule their art experience sessions conveniently.

  • Seamless Online Payments: Implemented Xendit API as a payment gateway, enabling secure online transactions for customers (Xendit account setup advised for the client).

  • Content Management System (CMS): Provides Cleverlli with an admin dashboard to manage product categories (painting game variations), track cash flow, and schedule appointments efficiently.

  • Responsive Design: Ensures optimal viewing and functionality across desktops, tablets, and mobile devices.
